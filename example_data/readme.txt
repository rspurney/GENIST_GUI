Example Data

Note that this data is for Arabidopsis thaliana.

Use GENIST_* files to run the GENIST tab:
	Genes File: GENIST_genesfile.xlsx (DS8)
	Time Course File: GENIST_timecoursefile.xlsx (DS3)
	Clustering File: GENIST_clusteringfile.xlsx (DS6)

Use RTPSTAR_* files to run the RTP-STAR tab
	Genes File: RTPSTAR_genesfile.xlsx (DS14)
	Gene Expression File: RTPSTAR_expressionfile.xlsx (DS15)
	Clustering File: RTPSTAR_clusteringfile.xlsx (DS16)
	Time Course File: RTPSTAR_timecoursefile.xlsx (DS17)
